#Google Map App by ZhangJian

##How to run this App
* Open index.html in your browser then the application will run.
* Click the arrow icon on top-left of the screen you can display or hide the list view.
* You can filtrate the result by typing in the search bar.
* You can either click the list or click the marker icon to choose a location.
* When a location is chosen, the InfoWindow will display an image of this location provided from flicker.

##Functionality of the app includes:
* An interactive navigation bar to filter map and navigation bar results using knockout.js.
* Google Map API to display the map, markers and InfoWindow.
* Flicker API to get image by latitude and longitude.
